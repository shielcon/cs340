This is a sample CS340 Project website based on the BSG Database.

Menu of operations

  • List, Add, Update and Delete Persons (perform Create, Read, Update, Delete operations on a single entity)
  • Change Certifications for a Person (change M-to-M relationships)
  • Change Homeworld of a Person (change 1-to-M relationships)
  • List and add new planets
  • List and add new certifications
  • List and add new spaceships
  • Search people by Last name
  • Filter people by Certifications they have
