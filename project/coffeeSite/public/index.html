Final project for CS340 - Coffee Database

This website is used to view and modify a database storing records of coffee.

Menu of operations

  • List, Add, Update, and Delete Coffees
  • List, Add, and Delete Roasters
  • List, Add, and Delete Origins
  • List, Add, and Delete Varieties
  • Show Coffees by Roast (one-to-many relationship)
  • Show Coffees by Roaster (one-to-many relationship)
  • Search coffees by name
